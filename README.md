# Restaurant Reviews

## Install
```
npm install
```

## Launch

* Webpack Dev Server

Open http://localhost:8000/ in a browser after having launched the following command:
```
npm run serve
```

* Build

Open http://localhost:8000/ in a browser after having launched the following command:
```
npm run build
cd dist && python -m SimpleHTTPServer
```

## Third-party API dependencies
[Google Maps](https://developers.google.com/maps/)
